# Instructions pour le Logo

Pour ajouter le logo WinPawa :

1. Copiez votre fichier logo (le fichier PNG avec le "W" doré) dans ce dossier
2. Renommez-le en `logo.png`
3. Le logo sera automatiquement utilisé dans :
   - Le header (navigation)
   - La page de login/register
   - L'icône PWA

Le logo doit être au format PNG avec fond transparent pour un meilleur rendu.

Dimensions recommandées : 400x400px ou plus
